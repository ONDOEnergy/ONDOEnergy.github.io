# ONDO Energy - Design Transformation Presentation

## Overview

This presentation showcases the complete UI/UX redesign of the ONDO Energy platform, comparing the old and new designs side-by-side using an interactive slider interface.

![Banner](../banner.png)

*Desktop and Mobile Design Showcase*

## Features

- **Interactive Before/After Slider**: Drag to compare old and new designs
- **PDF Export**: Generate professional PDF presentations for email sharing
- **Responsive Design**: Fully optimized for desktop, tablet, and mobile devices
- **Modern UI/UX**: Beautiful interface with smooth animations and transitions

## Technical Details

- Built with HTML, CSS, and JavaScript
- Uses Tailwind CSS for styling
- jsPDF for PDF generation
- Custom before/after slider implementation
- Fully responsive and mobile-optimized

## Usage

Simply open `index.html` in a web browser to view the presentation. Use the slider controls to compare old and new designs, and click "PDF" to export a presentation-ready document.

## Interactive Demo

The presentation includes an interactive before/after slider on each page. Drag the slider handle to reveal the old design on the left and the new design on the right, allowing for a seamless visual comparison.
